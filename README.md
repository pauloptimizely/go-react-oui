## Go React OUI Generator

```
npm install -g go-react-oui
```

### CLI Usage

**Args**

- `c` - component name
- `d` - component's base directory

```
g react -c Button -d ./src
```

This will generate a `Button` pure component in the currect working directory's src folder

```
src/
  Button/
    Button.js
    Button.spec.js
    index.js
```
